Score & Temperatura

Sistema de priorização 0–100

Todo lead entra com um score automático. A temperatura define a urgência de contato e a fila de distribuição dos SDRs.

🔵

Fila normal / sem distribuição

Critérios de pontuação Automático

→ Faturamento acima de R$5M → +30 pts
→ Faturamento R$1M a R$5M → +25 pts
→ Time acima de 20 pessoas → +15 pts
→ Descrição do negócio detalhada → +20 pts
→ "Quero aplicar imediatamente" → +20 pts
→ Pagou taxa de aplicação (R$5K) → score máximo

Distribuição por temperatura Round-robin

→ Fervendo + Quente → pool único, round-robin
→ Quem tem menos leads recebe o próximo
→ Morno → round-robin separado
→ Frio / Gelado → sem distribuição ativa
→ SDR notificado via WhatsApp com resumo do lead
→ Carta coringa: pagou aplicação → fundador ou SDR sênior
Automações

8 automações mapeadas end-to-end

O agente instrui como montar cada automação no HubSpot — gatilho, ação e resultado esperado.

Entrada de Lead Gatilho: formulário

→ Validar CPF / CNPJ / WhatsApp
→ Calcular score automaticamente
→ Criar card com todos os dados
→ Atribuir tag de origem
→ Distribuir para SDR por temperatura
→ Notificar SDR no WhatsApp

Agendamento de Call Gatilho: Calendly

→ Criar evento no Google Meet
→ Confirmar agendamento via WhatsApp
→ Gerar briefing via Claude IA
→ Inserir briefing no card
→ Notificar Closer com script
→ Lembrete automático D-1 e 2h antes

Fechamento Gatilho: closer fecha

→ Gerar contrato via Autentic
→ Enviar para assinatura do cliente
→ Após assinatura → mover para CS
→ Criar grupo WhatsApp de onboarding
→ Enviar kit de boas-vindas
→ Agendar call de onboarding com CS

Follow-up Passivo Gatilho: tempo sem interação

→ D+2: mensagem de reengajamento (prova social)
→ D+7: conteúdo relevante ao segmento
→ D+15: oferta especial ou novo gatilho
→ D+30: última tentativa
→ Sem resposta → move para Perdido

No-show Gatilho: hora da call passou

→ Move card para No-show automaticamente
→ SDR notificado para reagendar
→ Tentativas D+0, D+2, D+5
→ Reagendou → volta para Agendado
→ Não reagendou → move para Follow-up

OKR Diário Gatilho: 23:59

→ Calcular ligações do SDR via Just Call
→ Calcular calls do Closer via Calendly
→ Atualizar OKRs no ClickUp
→ Enviar resumo ao Head Comercial no WhatsApp
Treinamento de Time

Protocolos para SDR e Closer

O agente instrui cada papel do time com protocolos claros de uso do CRM, o que registrar e quais OKRs perseguir.

🧑‍💼 SDR — Protocolo completo

Responsável pelas etapas 1 a 4. Objetivo: qualificar e agendar — nunca fechar.

1 Abrir card e ler score + temperatura do lead
2 Analisar dados do formulário + Instagram + histórico
3 Enviar mensagem personalizada — nunca template genérico
4 Registrar resultado no card: respondeu / não / inválido
5 Sequência de 4 contatos (D+0, D+1, D+2, D+4)
6 Passar briefing completo ao Closer antes de transferir
50–100 contatos/dia OKR SDR
≥ 40% dos qualificados agendados Meta agendamento

🤝 Closer — Protocolo completo

Recebe leads qualificados com briefing pronto. Faz a call e conduz ao fechamento.

1 Ler briefing gerado pela IA no card antes da call
2 Ver histórico de conversa do SDR
3 Abrir script NEPIC personalizado (gerado automaticamente)
4 Dividir telas: lead na esquerda, script na direita
5 Fireflies ativado — grava e transcreve automaticamente
6 Registrar objeções, decisão e próximo passo no card
5 calls/dia mínimo OKR Closer
≥ 25% das calls → fechamento Meta conversão
